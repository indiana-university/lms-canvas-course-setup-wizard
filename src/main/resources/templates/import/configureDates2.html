Canvas will use the dates you provide to distribute event and due dates across the term for the current course.

Recommendations:

  • Same term lengths: (e.g. both courses are 16 week, 8 week, etc.), enter the first day of class for both classes and leave the last day of class empty for both. Use the actual start date for both classes, not the term start date.
  • Different term lengths: Enter first and last day dates for both courses, using the actual first and last date of class, not the term start and end dates.
