You have opted to select the theme (page design) and banner image for this course.

Note: This option should only be used when creating new course content. If you want to apply these design elements to existing course materials, contact your campus teaching and learning center for assistance.

Once you have confirmed your choices, the wizard will generate a course home page and a syllabus design as well as templates for creating assignments, discussions, quizzes, and pages, all with visual design elements that you chose.
